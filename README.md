# Basic_Planner_Project
일정 관리 프로그램
## 👩🏻‍💻 프로젝트 선정 배경
일정을 효율적으로 기록하고 관리하는 개인용 플래너가 필요하여 기획하게 되었습니다.
 <br>
 
## 📅 개발 일정
25.10.30 ~ 25.12.4
<br>

## ⚙️ 주요 기능
1) 일정 관리 : 일정 이름, 날짜, 시간, 카테고리, 중요도, 완료 여부 등을 구조체로 관리
2) 일정 조회 : 전체 목록 출력,특정 학생 검색(이름,학번)
3) 일정 수정 및 삭제 : 기존 데이터 수정 또는 삭제
4) 통계 기능 : 완료율, 날짜별 일정 수 확인 가능
5) 파일 저장/불러오기 기능 : 각 일정 요소들을 한번에 확인 가능하도록 저장/불러오기
<br>
